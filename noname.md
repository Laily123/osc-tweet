几种使用方式
```shell
// content from tuning
osc-tweet -u xxx-name -p xxx-password -o joke
// "your-message" is the tweet content
osc-tweet -u xxx-name -p xxx-password -o tweet -m your-message 
// config-path preferably using full path, config file example: config.samle.ini
osc-tweet -c config-path 

// osc-tweet -h for more help
```
